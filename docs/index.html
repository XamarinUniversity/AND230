Before you begin

This is an Android class which will require a Xamarin.Android development environment to complete the coding exercises. Please read the setup requirements if you need help installing the Xamarin development environment.

In addition, you will need to install the optional Google Play Services SDK library from the Android SDK Manager and get an Android emulator with Google Play Services installed, or (preferably) a device to run the exercise on.

Testing on an emulator

Testing an app using Google Maps on an emulator requires that emulator having Google Play Services installed on it. You can install Google Apps on your emulator whether using a Google Android emulator, Xamarin Android Player or Visual Studio's Android Emulator by following the online instructions.

Using Google's pre-built Android image

If you want to just run an image with the APIs pre-installed, then the best approach is to download the Google emulator image (remember to use x86 for performance). This is done using the Android SDK Manager. Make sure that you download the Google API package for the same API version as the emulator that you will be using.

Create a new emulator image or edit an existing emulator by clicking on the Open Android Emulator Manager in the Tools menu option in Visual Studio. Make sure to set your emulator's Target to a Google API support target API level, which provides you with an emulator with Google Apps already installed.

Now when you deploy to an emulator, choose the emulator with Google APIs loaded, and Google Maps APIs will work in your emulator.

Testing on a physical device

You can tell whether maps are supported by the existence of the Google Settings app, and possible the Google Maps app on the device. If either of these apps are present, then the Google APIs are available, otherwise they are not. There is no way to add Google APIs to an unsupported device today - it must be included by the OEM vendor.

Overview

In this course, we will define what is Google Play Services, install and use it on an emulator. We will discuss the major Google Maps API V2 classes to display mapping data in your Android application.

Objectives

  1. Explore Android map options
  2. Configure an application to use Google Maps
  3. Display a map in your application
  4. Add markers to a Google Map
  5. Adjust the map position and perspective

Exercises

    Exercise
    In this exercise, you will show the initial map and set the map type to change its style.

    Exercise
